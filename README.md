
# Kiểm thử Unit Test cho Bài tập lớn OOP

Bài tập lớn: Hệ thống quản lí thư viện Objectum

Lớp: 2425I_INT2204_17
   

## Features

- Mượn, trả, thêm xóa sách
- Phân biệt Member và Admin
- Nhạc trong ứng dụng
- API để tìm sách
- API để tìm kiếm nhạc


## Tech Stack

- Java, JavaFX
- MySQL
- Google Books APIs
- Jamendo API
- URL các trò chơi web
